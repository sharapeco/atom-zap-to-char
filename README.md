zap-to-char
================================================================

A minimum zap-to-char implement for Atom Editor.

Difference from Emacs/xyzzy is ...

* Zap to string, not only char.
* Can only zap to end of line.


Default Keymap
----------------------------------------------------------------

```
"atom-text-editor:not([mini])":
	"alt-z": "zap-to-char:zap-to-char"
```
